Record and linkage visit_number Visit Number A number that identifies the visit or timepoint of data collection (baseline=1 and each subsequent follow up is 2 or greater). integer
Record and linkage shifted_visit_dt Visit Date (shifted) This is the date of the visit (ie time point) after shifting by a random number 182 days before or 182 days after the actual visit date. The shifting is done on the individual level which retains the time interval between visits (and allows calculations of days between time points or other date fields which are shifted by the same number of days). To avoid issues with type conversion errors, the date is of type integer date %Y%m%d
